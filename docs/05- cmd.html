Les commandes termiales

Premier aperçu

Il est possible de démarrer l'interface à l'aide du terminal de commandes. Pour cela, veuillez utiliser les paramètres et options suivantes. Lorsque vous demandez de l'aide, voici le panneau d'information

usage: main.exe [-h] [-p PROJECT [PROJECT ...]] [-pdf PDF [PDF ...]] [-img IMG [IMG ...]] [-sea SEA [SEA ...]]

[-csv CSV [CSV ...]] [-run MODE] [-exe PROGRAMME FONCTION] [-o OUTPUT] [-exit] [-save]

Logiciel de traitement de fichiers PDF. Il est possible de contrôler l'ouverture du programme en passant ici par la

ligne de commande. Vous pouvez utiliser plusieurs fois chaque commande. Cependant, à partir du deuxième projet ouvert,

tout sera importé et non ouvert.

options:

-h, --help show this help message and exit

-p PROJECT [PROJECT ...]

Le projet s'il y en a un à ouvrir

-pdf PDF [PDF ...] Ouvre dans le logiciel un fichier, de type PDF

-img IMG [IMG ...] Ouvre dans le logiciel un fichier, de type image

-sea SEA [SEA ...] Importe une fonction et lance le programme d'enregistrement

-csv CSV [CSV ...] Ouvre les fichiers dans une liste de fichier au format csv ou excel

-run MODE Lance un programme du mode basic : FUSION, IMAGES, CSV

-exe PROGRAMME FONCTION

Lance le programme indiqué par la fonction. Syntaxe : [programme fonction]

-o OUTPUT Fichier ou dossier de sortie pour l'éxécution du programme

-exit Ferme la fenêtre une fois l'action réalisée

-save Enregistre le projet ouvert (s'il y en a un) automatiquement avant de fermer

Merci d'utiliser notre application !

Ainsi, pour lancer une commande, utilisez la syntaxe suivante

main.exe -h

qui permet d'ouvrir la page aide dans le terminal.

Importation de fichiers

Vous pouvez demander l'ouverture d'un fichier PDF depuis cette ligne de commande, à l'aide de

main.exe -pdf nom-du-fichier.pdf "nom-du-fichier 2.pdf"

Pour chaque attributs de fichiers, vous pouvez en mettre plusieurs. C'est-à-dire, vous pouvez ouvrir dix fichiers différents depuis la ligne.

Vous pouvez importer des fichiers CSV ou Excel de la même manière, avec l'attribut -csv.

De même pour les images et les fonctions du logiciel.

Attention, pour les projets, lorsque vous en importer plusieurs, le premier est ouvert normalement, mais les suivants sont IMPORTES dans le premier !

Paramètrage du logiciel

Vous pouvez ensuite choisir le dossier de sortie du programme (en bas dans la fenêtre). Pour cela, indiquez le paramètres -o nom-du-fichier.pdf ou bien -o nom-du-dossier

Le logiciel détecte si vous avez mis un dossier ou un fichier automatiquement.

Pour lancer une action

Pour lancer une action, vous pouvez utiliser les paramètres -run et -exe.

Paramètre -run

Le paramètre -run propose différentes fonctions par défaut

- avec CSV vous pouvez exporter la liste de fichier sous un format .csv, dans le fichier indiqué par le pramètre -o.

- avec IMAGES vous pouvez exporter l'ensemble des images dans le dossier indiqué par le paramètre -o.

- avec FUSION, vous pouvez exporter la fusion générale des fichiers dans le fichier de sortie indiquée par le paramètre -o.

Paramètre -exe

Le paramètre -exe permet de lancer un programme automatiquement.

Pour cela, faites suivre le -exe du nom du programme (ex. ArrangerDuo, ArrangerTrio, SplitFile, RognerFile, OrganizePages, ...), suivit du fichier de la fonction à éxécuter (ex. ./programs/Program-31.pdfsea)

Le logiciel lance automatiquement l'éxécution du programme en question, qui utilise les paramètres indiqués par l'argument -o. Si aucun n'est donné, alors il enregistre dans le fichier par défaut.

Fin de l'éxécution

Enfin, une fois l'action réalisé, vous pouvez demander de fermer la fenêtre : pour cela, ajoutez le paramètre -exit dans la commande.

Si vous souhaitez enregistrer le projet avant de le fermer, pensez à utiliser l'attribut -save qui enregistre le projet ouvert, et si aucun n'est ouvert, propose de l'enregistrer sous.
