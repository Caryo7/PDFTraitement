Documentation du logiciel simplifiée

Le logiciel de traitement de PDF peut être utilisé de manière simple, sans avoir à utiliser tous les paramètres et programmes. Il peut ainsi permettre la fusion de fichiers, et l'extraction de certaines pages d'un fichier.

Ouvrir des fichiers

Le logiciel peut être utilisé de manière simple, sans avoir à entrer dans les détails pour l'utiliser. Pour cela, lancer le programme. Par le menu "Fichier", cliquez sur importer des fichiers et choisissez un ou plusieurs fichiers. Vous pouvez aussi importer une liste de fichiers pour aller plus vite. De la même manière, il est possible de mettre un ou plusieurs fichiers par glissé déposé.

Une fois le ou les fichiers déposés, vous voyez dans un onglet la liste des fichiers, avec : le nom du fichier dans une colonne, la taille du fichier, le nombre de pages, si il faut inverser l'ordre, et les pages à conserver. Pour éditer ses deux derniers paramètres, double cliquez sur le nombre et entrez le nouveau.

Dans le deuxième onglet, vous avez accès à un aperçu de la liste des pages.

Extraire facilement des pages

Vous pouvez extraire facilement des pages à partir de l'onglet Pages. Pour cela, séléctionnez les différentes pages qui vous intéresse, puis, en cliquant sur clique Droit, séléctionnez "Extraire des pages séléctionnées". Le logiciel vous demande dans quel fichier enregistrer ces pages. Une fois les pages exportées, elles sont désélectionnées automatiquement. Vous pouvez exporter aussi une unique page en cliquant sur "Extraire cette page".

Exporter les fichiers

Vous pouvez exporter tous les fichiers dans la liste (soit toutes les pages dans la l'onglet pages) en allant dans le menu "Fichier" et en cliquant sur "Exporter la fusion".

Il est possible d'exporter la liste des fichiers présents en cliquant sur "Exporter la liste".

Enfin, si vous le souhaitez, vous pouvez exporter l'ensemble des images en une haute résolution dans un dossier en cliquant sur "Exporter les images".

Utilisation complète des fonctions

Le logiciel est architecturé en plusieurs programmes, qui ont chacun plusieurs fonctions. Les programmes sont fourni avec le logiciel et ne peuvent être reprogrammés (sauf accès directe au code source).

Nous détaillerons ici d'abord les fonctions communes à tous les programmes, puis tous les programmes disponibles.

Voici un bref aperçu de la fenêtre principale.

Les fonctions

Elles sont disponible sous la forme de liste de conditions. Pour y accéder, cliquez sur un programme, comme Extraire une page, puis, séléctionnez une fonction dans la liste et cliquez sur modifier. Vous voyez alors la fenêtre d'édition d'une fonction. Dans la liste, vous pouvez remarquer trois colonnes : une pour la priorité de la condition, puis une autre pour la condition et enfin, une dernière pour le (ou les) résultat(s) à sortir.

Il est possible d'ajouter une condition en cliquant sur le bouton en question, et d'en enveler. Vous pouvez aussi en dupliquez.

Chaque condition doit donc posséder une condition reposant sur deux paramètres : la page en cours de traitement (

i
), et le nombre de pages totales (
l
).

Pour cela, vous devez impérativement avoir une équation de la forme

A = B
pour que la sortie soit validée.

Par exemple, vous pouvez avoir

i % 2 = 0
comme condition, ce qui revient à dire que
i
est pair,

i; l - (1 - i)
comme sortie, ce qui indique que sur la page de résultat, il faut mettre à gauche la page actuelle, et à droite la page dont l'ordre par de la fin.

Les indications sur les paramètres donnés en condition et en sortie sont disponibles dans l'onglet Informations, accessible depuis le bas de la fenêtre.

Les programmes

Voyons maintenant les programmes disponibles et leur fonctions par défaut installées.

Remarque: au début de chaque éxécution, tous les fichiers dans la liste sont fusionnés selon leurs paramètres de pages et d'inversion, et que le traitement se fait sur cette fusion.

Extraire des pages

Le programme Extraire des pages permet de ne conserver que certaines pages dans un fichier. Vous pouvez ainsi ne conserver que les pages paires, ou qu'une page sur trois, ou que la première page, ...

Il possède deux fonctions par défaut, très simples :

- Extraction d'une page sur deux

- Extraction d'une page sur trois

Vous pouvez créer vos propres fonction qui doivent avoir une sortie de la forme

1
: garder la page

0
: ne pas garder la page.

Extraire sous forme d'images

Ce programme permet d'extraire des images du fichier.

Il possède trois fonctions par défaut :

- Extraire toutes les images

- Extraire une page sur deux

- Extraire une page sur trois

Vous pouvez créer vos propres fonctions, qui doivent avoir deux arguments en sortie, séparés par des "

;
".

Ces arguments sont

1
ou
0
pour garder ou non la page, et la résolution en DPI pour l'extraction.

Attention, plus la résolution est importante, plus le temps nécéssaire pour l'extraction l'est aussi !

Remarque: par défaut, la résolution est de 330 DPI
