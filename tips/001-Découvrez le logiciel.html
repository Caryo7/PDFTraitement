Premiers pas

Vous pouvez utiliser le glisser-déposé pour mettre un fichier dans la liste.

Plusieurs fichiers peuvent être mis en même temps, d'extensions différentes.

Les extensions sont les suivantes :

o .pdf pour les fichier Portable Document Format (ou .pdff)

o .xlsx pour les fichiers Excel, liste de fichier PDF

o .csv table pour les liste de fichier PDF

o .png pour toutes les images (.jpg, .gif, .tiff, ... sont aussi autorisées)

o .pdfsea pour les composants programme du logiciel

o .pdfpro pour les projets
